# Experimento: Comparação de LLMs na Geração de Planos de Aula

Este repositório contém o design experimental para comparar dois modelos de linguagem (LLMs) na geração de planos de aula para ensino de programação utilizando *example-based learning*.

## Objetivo

Avaliar a qualidade pedagógica dos planos de aula gerados por dois modelos distintos.

## Estrutura

* `design_experimental.md` → visão geral do experimento
* `tarefas.md` → definição das tarefas utilizadas
* `rubrica.md` → critérios de avaliação
* `analise.md` → plano de análise dos resultados

## Metodologia

* Mesmos prompts para ambos os modelos
* Avaliação cega
* Uso de rubrica padronizada
* Análise estatística dos resultados

## Observação

Todos os artefatos necessários para replicação devem ser disponibilizados neste repositório.
